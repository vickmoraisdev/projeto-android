🌐 Projeto - Desafio 10 Curso em Vídeo
Este site foi desenvolvido como parte de um desafio proposto pelo professor Gustavo Guanabara, do Curso em Vídeo.

🚀 Sobre o projeto
Durante a criação deste site, tive a oportunidade de desenvolver novas habilidades e explorar melhor o uso do CSS, que antes era uma das minhas maiores dificuldades. Fiz quase 100% do projeto sozinha, teve algumas pequenas partes que precisei de ajuda, pois não tinha aprendido como fazia.
Ao longo do projeto, consegui entender melhor conceitos importantes como:
📏 Largura (width)
📐 Altura (height)
🎨 Diferentes formas de estilização no CSS

Esses eram pontos que antes me deixavam confusa, mas aqui consegui aplicar na prática e melhorar bastante meu entendimento.

💡 Aprendizados
Melhor organização de código em HTML e CSS
Uso mais consciente das propriedades de estilo
Estruturação mais limpa e legível

Tentei manter o código o mais organizado possível, pensando tanto na legibilidade quanto na prática de boas práticas.
